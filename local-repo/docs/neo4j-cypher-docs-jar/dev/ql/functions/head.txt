[[functions-head]]
== HEAD ==
`HEAD` returns the first element in a collection.

*Syntax:* `HEAD( expression )`

*Arguments:*

* _expression:_ This expression should return a collection of some kind.

_Query_

[source,cypher]
----
START a=node(2)
RETURN a.array, head(a.array)
----


The first node in the path is returned.

.Result
[queryresult]
----
+---------------------------------------+
| a.array               | head(a.array) |
+---------------------------------------+
| ["one","two","three"] | "one"         |
+---------------------------------------+
1 row
0 ms

----
