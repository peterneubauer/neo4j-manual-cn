[[create-create-single-node]]
== Create single node ==
Creating a single node is done by issuing the following query.

_Query_

[source,cypher]
----
CREATE n
----


Nothing is returned from this query, except the count of affected nodes.

.Result
[queryresult]
----
+-------------------+
| No data returned. |
+-------------------+
Nodes created: 1
0 ms

----



.Try this query live
[console]
----
(1) {}


create n
----
