_Example request_

* *+POST+*  +http://localhost:7474/db/data/node/72/relationships+
* *+Accept:+* +application/json+
* *+Content-Type:+* +application/json+
[source,javascript]
----
{
  "to" : "http://localhost:7474/db/data/node/999999",
  "type" : "LOVES",
  "data" : {
    "foo" : "bar"
  }
}
----
